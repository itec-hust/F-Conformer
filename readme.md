# Frequency-Conformer (F-Conformer)

A PyTorch-based high-resolution piano transcription system.

---

## 1. Installation
```bash
pip install -r requirements.txt
```

## 2. Quick Start
```bash
bash runme_conformer.sh
```

## 3. Pre-trained Weights
You can download the pre-trained model weights (.pth files) from the following link:

[Download Pre-trained Models Here](https://github.com/itec-hust/F-Conformer/releases/tag/v1.0.0)
